Sobre mi

Mi nombre es Jorge Daniel De Anda Almanzar, me considero alegre, entusiasta, dedicado, responsable, con ánimos y la disposición de aprender cosas nuevas. Me gusta el desarrollo web Front-End y el desarrollo de aplicaciones móviles.

Tengo conocimientos básicos en desarrollo Web con HTML5, CSS3, JavaScript, React.js y React Native todos estos con un nivel medio.

Servicios

Diseño Web Front-End

Conocimiento en el desarrollo y diseño de páginas web interactivas empleando el uso de HTML 5 para la estructuración y planificación del esqueleto de la página web, CSS 3 para el diseño y aspecto visual de la página, Java Script para la interacción de la página con el usuario.

Diseño responsivo

Conocimientos en la creación de páginas web adquiriendo un diseño con vistas responsivas utilizando librerías de apoyo para su visualización correcta en distintos dispositivos con acceso a internet.

Desarrollo Móvil

Conocimiento básico en realización de aplicaciones móviles mediante el framework React Native que comprende el desarrollo de la interfaz de usuario, así como, un poco de conocimeinto en Adobe XD.
